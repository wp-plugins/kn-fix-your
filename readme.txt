== Installation ==

= Installation =

1. Upload the file `kn-fix-your-title.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Please change the settings and enjoy!

= Updating =

If you had a trouble in updating this plugin, try deactive old version and delete the folder `kn-fix-your-title` from the `/wp-content/plugins/` directory. Then, try download new version and install as usual.
